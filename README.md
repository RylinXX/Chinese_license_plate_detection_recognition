# 🏗️ 智慧工地车牌识别与进出趟数统计系统

本系统是一套端到端、专为**工地/厂区出入口**设计的智能车牌识别与车辆运载趟数统计系统（轻量化云端方案原型）。
前端采用 Vanilla CSS/JS 打造极具视觉冲击力的**深色毛玻璃科技感大屏**，后端基于 **FastAPI + SQLite** 实现高并发接收、5分钟去重防抖过滤、每日趟数聚合统计、历史账目人工补录及报表一键导出。

---

## ✨ 核心业务特色

1. **科技感暗黑毛玻璃大屏**：采用现代 UI 设计语言，集成了今日进场、出场及场内滞留 KPIs 指标卡。
2. **历史通行检索中心**：支持点击日期选择器，一键切换并查阅任意历史天数的车牌通行数据与趟数统计。
3. **每日车辆运载趟数榜 (一天一统计)**：自动按车牌号分组计算车辆的进出次数，并以出场（OUT）装土次数作为该车“累计运载趟数”进行高颜值排行榜展示，每天零点自动重置。
4. **✍️ 手工记账与历史补录**：支持管理员手动输入车牌、颜色、方向。补录时间会自动绑定当前大屏选中的日期，完美解决了纸质账本“正”字统计向系统补录的工程痛点。
5. **🔍 限流流展示与磨砂大弹窗**：右侧实时记录流默认限流展示最新的 **前 5 条**，底部集成大按钮，点击一键唤醒毛玻璃大模态弹窗，以数据大表详细查阅该日所有的通行全日志，保持大屏的极简与专注。
6. **云端/本地算法适配器**：默认使用本地加载 of 轻量 PyTorch 目标检测与多任务识别网络（本地模拟云端），并预留了阿里云/腾讯云 OCR 车牌识别接口，随时支持零维护一键上云。

## 🚀 三步极简开箱部署

系统已在本地完美联调成功，并灌入了 **120 条高保真演示数据**。你可以直接通过以下步骤在本地跑起来：

### 步骤 1：激活虚拟环境并启动后台服务
打开终端，进入项目根目录下，运行以下命令启动 FastAPI 服务器：
```bash
# 激活项目自带的虚拟环境 (Windows PowerShell/CMD)
.venv\Scripts\activate

# 启动 Web 服务器
python web-server.py
```
*启动成功后，终端会提示：`Uvicorn running on http://127.0.0.1:8000`*

### 步骤 2：浏览器打开智慧大屏看板
在浏览器（推荐 Chrome 或 Edge）中输入以下地址，即可大饱眼福地查阅数据看板：
👉 **[http://127.0.0.1:8000](http://127.0.0.1:8000)**

### 步骤 3：数据清空与重新灌入演示数据（选做）
如果你想重新体验清空并灌入演示数据的过程，系统提供了一键灌入脚本。在虚拟环境下执行：
```bash

python scratch/generate_demo_data.py
```
*执行后，会瞬间在 SQLite 中重新生成 120 条逻辑合理、进出交替的今天/昨天/前天高保真演示通行流。*

## 🛠️ 工地生产环境物理联调与上云指引

当你决定将这套系统正式布线并架设到工地上时，请参阅以下实操步骤：

### 1. 门口高清网络摄像头（IPC）如何联动？
市面上绝大多数商用智能车牌识别相机（如海康威视、大华、臻识等）都自带补光、防水及嵌入式车牌识别。你只需进入摄像头的 Web 后台管理页面：
1. **配置 HTTP 抓拍推送**：将推送 URL 目标指向你服务器的 API 地址：
   * **出口相机推送地址**：`http://你的服务器IP:8000/api/upload?direction=OUT`
   * **入口相机推送地址**：`http://你的服务器IP:8000/api/upload?direction=IN`
2. **效果**：当工程车驶过，相机识别成功后会快门抓拍图片并 POST 给我们的后端，大屏在不刷新的情况下就会实时滑入最新数据、重绘趟数排行榜！

### 2. 如何免维护无缝升级到“公网云端 OCR API”？
我们已经在 [recognizer.py](file:///c:/Users/scodi.KYLINX/Documents/车牌识别系统/recognizer.py) 中通过抽象基类设计好了标准接口。
1. 在 `recognizer.py` 的 `TencentCloudRecognizer` 适配器中填入你在腾讯云后台购买的 API `SecretId` 与 `SecretKey`。
2. 打开 [web-server.py](file:///c:/Users/scodi.KYLINX/Documents/车牌识别系统/web-server.py)，将初始化识别引擎的一行：
   ```python
   # 默认：加载本地 PyTorch 进行本地识别（模拟云端）
   recognizer = LocalSimulatedCloudRecognizer(...)
   ```
   **修改并替换为**：
   ```python
   # 升级：一键无缝接入腾讯云商用 OCR 车牌识别服务，免去本地 GPU 算力负担
   recognizer = TencentCloudRecognizer(secret_id="你的SecretId", secret_key="你的SecretKey")
   ```
3. 数据库和前端大屏逻辑完全不需要做一字修改，平滑上云，实现边缘端零硬件算力开销！
